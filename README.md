# test-task

Vue test-task application with possibility of saving data to nested table in local storage. [Element plus](https://element-plus.org/en-US/) design system used. 

<br>
Stack: 
<br>
<ol>
<li>Vue</li>
<li>Vuex</li>
<li>Local storage</li>
<li>Element plus design system</li>


</ol>
